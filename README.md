# بوت تيليجرام لتنزيل فيديوهات TikTok وInstagram وYouTube

هذا مشروع بوت تيليجرام مكتوب بلغة Python. يستقبل البوت رابط فيديو من TikTok أو Instagram أو YouTube، ثم يستخدم مكتبة `yt-dlp` لتنزيل أفضل جودة متاحة من المصدر وإرسال الفيديو للمستخدم داخل تيليجرام.

## تنبيه مهم حول العلامة المائية والحقوق

هذا المشروع مخصص لتنزيل المحتوى الذي تملكه أو لديك إذن صريح بتنزيله. الكود لا يتجاوز DRM، ولا يكسر حماية المنصات، ولا يلتف على قيود الوصول، ولا يزيل العلامات المائية بطرق غير مصرّح بها. بالنسبة إلى TikTok، قد تكون بعض المقاطع متاحة من المصدر بصيغ مختلفة؛ إذا أتاح المصدر نسخة نظيفة رسميًا فقد يستطيع `yt-dlp` جلبها، أما إذا كانت العلامة المائية جزءًا من الفيديو المتاح فلا يقوم هذا البوت بإزالتها اصطناعيًا.

## المميزات

- يدعم روابط TikTok وInstagram وYouTube.
- يحاول تنزيل أفضل جودة متاحة تلقائيًا.
- يدمج الفيديو والصوت في MP4 عند توفر `ffmpeg`.
- ينظف الملفات المؤقتة بعد الإرسال.
- يرفض الملفات التي تتجاوز حد الحجم المحدد في `.env`.
- رسائل عربية واضحة للمستخدم.

## المتطلبات

- Python 3.10 أو أحدث، ويفضل Python 3.11.
- توكن بوت تيليجرام من BotFather.
- تثبيت `ffmpeg` للحصول على أفضل دمج وجودة للفيديو والصوت.

على Ubuntu أو Debian يمكنك تثبيت ffmpeg بالأمر التالي:

```bash
sudo apt update
sudo apt install -y ffmpeg
```

## طريقة الحصول على توكن البوت

افتح تيليجرام وابحث عن `@BotFather`، ثم أرسل الأمر:

```text
/newbot
```

اتبع التعليمات، وبعد إنشاء البوت سيعطيك BotFather توكن يشبه هذا الشكل:

```text
123456789:ABCDEF_your_token_here
```

احتفظ به ولا تنشره علنًا.

## التثبيت والتشغيل

ادخل إلى مجلد المشروع:

```bash
cd telegram_video_bot
```

أنشئ بيئة افتراضية:

```bash
python3 -m venv venv
source venv/bin/activate
```

ثبت المتطلبات:

```bash
pip install -r requirements.txt
```

انسخ ملف الإعدادات:

```bash
cp .env.example .env
```

افتح ملف `.env` وضع توكن البوت:

```env
TELEGRAM_BOT_TOKEN=ضع_توكن_البوت_هنا
MAX_FILE_MB=49
```

شغّل البوت:

```bash
python bot.py
```

بعد التشغيل، افتح البوت في تيليجرام وأرسل `/start` ثم أرسل رابط فيديو.

## التشغيل الدائم على سيرفر

يمكنك تشغيل البوت داخل `tmux` أو `screen`:

```bash
tmux new -s video-bot
cd telegram_video_bot
source venv/bin/activate
python bot.py
```

للخروج من tmux مع إبقاء البوت يعمل اضغط:

```text
Ctrl+B ثم D
```

للعودة لاحقًا:

```bash
tmux attach -t video-bot
```

## ملاحظات عن Instagram

بعض روابط Instagram قد تفشل إذا كان الفيديو خاصًا أو يتطلب تسجيل دخول. هذا الكود لا يستخدم حسابك ولا يتجاوز قيود الخصوصية. استخدمه فقط مع الروابط العامة أو المحتوى الذي لديك إذن للوصول إليه.

## ملاحظات عن YouTube

قد تكون بعض الفيديوهات كبيرة جدًا، خصوصًا عند تنزيل أفضل جودة. إذا تجاوز الفيديو حد `MAX_FILE_MB` فلن يتم إرساله عبر البوت. يمكنك رفع الحد إذا كان حساب البوت وبيئة التشغيل يسمحان بذلك، لكن تيليجرام لديه قيود على حجم الملفات المرسلة عبر Bot API.

## تحديث yt-dlp

إذا توقفت بعض المنصات عن العمل، حدّث `yt-dlp` لأن المنصات تغير طرقها كثيرًا:

```bash
pip install -U yt-dlp
```
